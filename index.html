Bootstrap

$ git clone https://github.com/feejin/Silverstripe-CleanInstall.git
$ cd my-project
composer install
  • This is how projects start - our own bootstrap. Modules, gulp, blank theme.
  • It's an ever-shifting setup that we continually improve
  • Streamlines setup by putting as much in place as possible without being overly opinionated
  • It relies on some environmental stuff so don't copy it, but please do see if there's anything that might inspire you.

Composer.json

{
    "require": {
        "silverstripe/cms": "~3.1.15",
        "silverstripe/framework": "~3.1.15",
        …
    },
    "autoload": { "psr-4": { "Bigfork\\": "util/Bigfork" } },
    "scripts": {
        "post-install-cmd": "Bigfork\\Installer\\Install::postInstall",
        "post-update-cmd": "Bigfork\\Installer\\Install::postUpdate"
    }
}
  • Tilde explanation
  • As well as framework and CMS we're loading modules, talk about that next
  • Autoload and scripts
  • Config info for silverstripe database credentials, theme name and package.json setup
  • Configuration through _ss_environment so never any issues with editing config files that need to live in the repo
